# \* 주제 선정 및 타겟 사용자

## 주제 선정 목적

여행을 계획하는 과정에서는 여행지 선정, 일정 구성, 준비물 확인, 여행 시 유의사항 등을 각각 별도로 검색해야 하는 번거로움이 있다.

특히, 여행 경험이 부족한 사용자는 어떤 준비물을 챙겨야 하는지, 어떤 여행지가 자신의 조건에 적합한지 판단하기 어렵다.

본 과업은 사용자가 입력한 여행 정보를 기반으로 여행지 추천부터 여행 일정, 준비 체크리스트, 주의사항까지 자동으로 생성하여 여행 준비 과정을 효율적으로 지원하는 것을 목표로 한다.

## 타겟 사용자

* 국내·해외 여행을 준비하는 일반 사용자

* 가족 여행 계획자

* 커플 여행객

* 친구와 여행을 준비하는 사용자

* 출장 및 워케이션 이용자

## 1. 비교 목적

**추천 여행지 리스트 + 여행 준비 체크리스트 + 일정/예산/짐 목록**까지 여행 계획에 번거로움을 느끼는 사람을 위해 여행 관련 업무를 자동화하려 한다. 이 주제에 가장 적합한 LLM을 선정하기 위해,  동일한 입력 조건(시스템 프롬프트 + 테스트 케이스 3종)을 3개 모델에 투입하고, 사전에 정의한 평가 축 5개로 점수화하여 비교했다. "느낌"이 아니라 각 평가 축의 근거(실제 출력에서 관찰된 사실)를 기준으로 최종 모델을 선정하는 것이 목적이다.

## 5. 평가 기준

본 보고서는 동일한 프롬프트를 각 LLM에 입력한 후 아래의 7가지 평가 기준을 바탕으로 성능을 비교하였다.&#x20;

각 항목은 **5점 만점(1점: 매우 부족 \~ 5점: 매우 우수)​**으로 평가하였다.

| 평가 축                  | 평가 내용                                                             |
| --------------------- | ----------------------------------------------------------------- |
| **1. 정확성(사실/계산)​**    | 여행지 정보, 일정, 예상 비용, 준비물 등의 사실성이 높고 오류가 적은지 평가                      |
| **2. 한국어 자연스러움**      | 문장이 자연스럽고 읽기 쉬우며 사용자가 이해하기 쉬운 표현을 사용하는지 평가                        |
| **3. 긴 문맥(10턴) 유지**   | 대화가 이어질 때 이전 요구사항과 조건 및 금지 조건을 기억하고 일관성 있게 반영하는지 평가               |
| **4. 형식 준수(템플릿 출력)​** | 지정한 형식(여행지 추천, 일정, 체크리스트 등)을 정확하게 준수하는지 평가                        |
| **5. 속도/안정성**         | 응답 생성 속도가 빠르고 오류나 응답 중단 없이 안정적으로 결과를 제공하는지 평가                     |
| **6. 비용(요금제/토큰)​**    | 무료 사용 가능 여부, 사용량 제한, 토큰 효율 등을 고려했을 때 비용 대비 활용성이 높은지 평가            |
| **7. 환각 억제**          | 입력에 없는 수치나 데이터를 생성하지 않았는지 평가                                      |
| **8. 모호성 대응**         | 답변에 필요한 필수 정보가 부족할 때 추측하지 않고 확인 질문 등의 처리를 하여 질문에 대한 답으로 처리 했는지 평가 |
| **9. 도구 연동 적합성**      | API, 웹 서비스, 업무 자동화 도구와 연계하여 활용하기 적합한지 평가                          |

​

## 6-1. 평가 방법

* 동일한 테스트 프롬프트를 모든 LLM에 입력하였다.

* 동일한 입력 조건과 출력 형식을 사용하여 응답을 비교하였다.

* 각 평가 항목은 5점 만점으로 평가하였다.

* 총점은 40점 만점으로 산정하였다.

* 최종 선정 모델은 총점뿐 아니라 업무 자동화 적합성과 응답 품질을 종합적으로 고려하여 결정하였다.

## 8. 모델 비교 - 답변

## GPT - 5.5

​

좋은 조건이에요! \*\*“쇼핑 + 카페/디저트 + 예쁜 풍경 + 복잡하지 않은 동선”\*\*을 고려하면, 일본에서는 **후쿠오카**가 가장 잘 맞습니다.

​

## 맞춤형 추천 여행지: 일본 후쿠오카

### 왜 후쿠오카가 좋을까요?

* **공항에서 시내까지 지하철로 약 10분**이라 이동이 매우 편해요.

* 텐진, 하카타 중심으로 쇼핑몰과 카페가 모여 있어 **동선이 단순**합니다.

* 도쿄나 오사카보다 비교적 덜 복잡하고, **3박 4일 일정에 부담이 적어요.​**

* 겨울에도 많이 춥지 않은 편이라 여행하기 괜찮습니다.

* 예쁜 바다 풍경을 볼 수 있는 **모모치 해변, 오호리공원**도 가까워요.

​

## 추천 여행 스타일 요약

| 항목       | 추천                           |
| -------- | ---------------------------- |
| 여행지      | 후쿠오카                         |
| 추천 숙소 위치 | 하카타역 또는 텐진역 근처               |
| 주요 쇼핑    | 텐진 지하상가, 파르코, 캐널시티, 하카타역 쇼핑몰 |
| 카페/디저트   | 텐진, 다이묘, 하카타 주변              |
| 풍경       | 오호리공원, 모모치 해변, 후쿠오카 타워       |
| 이동 방식    | 지하철 + 버스 + 짧은 택시 이용          |

​

## 3박 4일 추천 일정

### 1일차: 도착 + 하카타 쇼핑

**동선: 공항 → 하카타 → 캐널시티**

* 후쿠오카 공항 도착

* 하카타역 근처 숙소 체크인

* 하카타역 쇼핑몰 구경

* 저녁에는 **캐널시티 하카타** 방문

**포인트:​** 첫날은 멀리 이동하지 않고 하카타 주변에서 가볍게 보내면 좋아요.

​

### 2일차: 텐진 쇼핑 + 카페

**동선: 하카타 → 텐진 → 다이묘**

* 텐진으로 이동

* 쇼핑 추천 장소

* 점심 후 다이묘 카페 거리 방문

* 예쁜 디저트 카페에서 휴식

* 저녁은 텐진 근처 맛집 또는 이자카야

**포인트:​** 텐진과 다이묘는 가까워서 쇼핑하다가 카페 쉬기 좋습니다. 너무 많이 걷기 싫다면 텐진 중심만 돌아도 충분해요.

​

### 3일차: 예쁜 풍경 + 여유로운 쇼핑

**동선: 오호리공원 → 모모치 해변 → 마크이즈 후쿠오카**

* 오전: **오호리공원** 산책

* 카페에서 휴식

* 오후: **모모치 해변 & 후쿠오카 타워**

* 근처 **마크이즈 후쿠오카 모모치**에서 쇼핑

**포인트:​** 풍경 보고 바로 쇼핑몰로 이동할 수 있어 동선이 깔끔합니다.

​

### 4일차: 마지막 쇼핑 + 귀국

**동선: 숙소 → 하카타역/공항**

* 하카타역에서 마지막 쇼핑

* 공항 이동

* 귀국

**추천 기념품**

* 로이스 초콜릿

* 도쿄바나나 계열 디저트

* 멘베이 과자

* 녹차/말차 과자

* 일본 드럭스토어 화장품, 파스, 감기약

​

## 예상 예산: 60만 원 기준

60만 원 예산이면 항공권 가격에 따라 조금 달라져요.가능하면 **저가항공 + 비즈니스호텔/게스트하우스급 숙소**를 추천합니다.

| 항목     | 예상 비용        |
| ------ | ------------ |
| 왕복 항공권 | 약 20만\~30만 원 |
| 숙소 3박  | 약 15만\~22만 원 |
| 교통비    | 약 3만\~5만 원   |
| 식비/카페  | 약 10만\~15만 원 |
| 쇼핑비    | 남은 금액 조절     |

### 예산 팁

* 숙소는 **하카타역 근처**로 잡으면 교통비와 시간을 아낄 수 있어요.

* 쇼핑 예산을 확보하려면 식사는 편의점, 라멘, 규동 등을 섞는 게 좋습니다.

* 카페는 하루 1곳 정도로 정하면 예산 관리가 쉬워요.

## 추천 숙소 위치

### 1순위: 하카타역 근처

* 공항 이동이 편함

* 쇼핑몰, 맛집, 교통이 모두 편리

* 첫 일본 여행이나 복잡한 동선을 싫어하는 사람에게 추천

### 2순위: 텐진역 근처

* 쇼핑 중심지

* 카페와 맛집이 많음

* 밤에 놀기 좋음

개인적으로는 이번 여행에는 **하카타역 근처 숙소**를 가장 추천해요.

## Claude Opus 4.8

​

안녕하세요! 저는 학습을 돕는 AI 튜터예요. 😊

말씀해주신 여행 계획은 정말 즐거워 보이지만, 저는 여행지 추천이나 여행 준비보다는 \*\*학습(개념 설명, 문제 풀이, 코드 리뷰, 디버깅 등)\*\*을 도와드리는 역할을 하고 있어요.

그래도 이왕 여행 이야기가 나왔으니, 학습과 연결해서 도와드릴 수 있는 방법들을 제안해볼게요!

### 이렇게 활용해보면 어때요?

1. **여행 계획을 프로그래밍으로 만들기**

   * 예산 계산기, 여행 일정 정리 프로그램을 파이썬으로 함께 만들어볼 수 있어요.

   python📋 복사

   * 이렇게 변수를 이용해 남은 예산을 계산하는 걸 배울 수 있어요!

2. **일본어 학습**

   * 쇼핑, 카페에서 쓸 유용한 일본어 표현을 함께 공부할 수 있어요.

3. **엑셀/데이터 정리 학습**

   * 여행 체크리스트를 표로 정리하는 방법을 배울 수 있어요.

혹시 위 주제들 중에 배워보고 싶은 게 있으신가요? 아니면 지금 공부하고 있는 다른 과목이 있다면 편하게 말씀해주세요! 함께 차근차근 도와드릴게요. 🌟

## Gemini 3.1 Pro

​

친구분과 함께하는 겨울 일본 여행이라니 정말 설레고 즐거운 시간이 될 것 같아요!

제시해주신 조건(쇼핑, 카페, 짧은 동선, 60만 원 예산)을 꼼꼼히 분석하여 가장 완벽한 맞춤형 여행지와 준비 체크리스트를 정리해 드립니다.

​

### ✈️ 추천 여행지: 일본 **후쿠오카 (Fukuoka)​**

도쿄나 오사카는 비행기 값과 숙박비가 비싸고, 지하철 노선이 복잡해 많이 걸어야 합니다. 반면 **후쿠오카**는 학생님의 모든 조건을 만족하는 최고의 선택지입니다!

**💡 추천 이유:​**

1. **짧고 단순한 동선 (걷기 최소화):​** 공항에서 시내(하카타/텐진)까지 지하철로 단 10\~15분 거리입니다. 쇼핑몰과 카페가 특정 지역에 밀집해 있어 길을 잃을 걱정이 없고 덜 걸어도 됩니다.

2. **예산 세이브:​** 일본 내에서 항공권과 숙박비가 가장 저렴한 편에 속해 60만 원 예산으로 방어하기 가장 좋습니다.

3. **쇼핑과 카페의 성지:​**

4. **겨울 풍경:​** 겨울철 하카타역 앞은 화려한 일루미네이션(빛 축제)으로 꾸며져 멀리 가지 않아도 무척 예쁜 풍경을 볼 수 있습니다.

​

### 💰 예산(60만 원) 활용 팁

3박 4일에 60만 원은 조금 타이트할 수 있으니 이렇게 계획해 보세요!

* **항공권:​** 특가 항공권을 노려 15\~20만 원 내외로 결제 (평일 출발 추천)

* **숙박비:​** 비즈니스 호텔 트윈룸을 친구와 반반 부담하여 1인당 10\~15만 원 내외로 해결

* **쇼핑/식비:​** 남은 25\~30만 원으로 가성비 좋은 쇼핑(GU, 돈키호테, 드럭스토어 등)과 예쁜 카페 디저트 즐기기

​

### 📝 여행 준비 체크리스트

성공적인 여행을 위해 꼭 챙겨야 할 항목들을 정리했습니다. 체크하면서 짐을 싸보세요!

**1. 필수 서류 및 결제 수단**

* **여권:​** 만료일이 6개월 이상 남았는지 꼭 확인하세요!

* **Visit Japan Web 등록:​** 입국 심사를 빠르게 통과하기 위해 출국 전 미리 큐알코드를 발급받으세요.

* **결제 수단:​** 트래블로그, 트래블월렛 같은 '해외 결제용 체크카드'를 발급받고, 현금(엔화)도 1\~2만 엔 정도 환전해 가세요. (작은 카페는 현금만 받는 곳이 있습니다)

**2. 쇼핑 맞춤 준비물**

* **접이식 보조 가방 (폴딩백):​** 쇼핑 후 짐이 늘어날 것을 대비해 캐리어에 꽂을 수 있는 가벼운 가방을 챙기세요.

* **면세용 여권 소지:​** 일본에서 쇼핑할 때 여권을 보여주면 면세(Tax Free) 혜택을 받을 수 있으니 외출 시 항상 챙기세요.

**3. 겨울 여행 & 편의 용품**

* **편한 운동화:​** 걷는 걸 최소화하더라도 쇼핑을 하다 보면 발이 피곤해집니다. 가장 편한 신발을 신으세요!

* **110V 어댑터 (돼지코):​** 일본은 우리나라와 콘센트 모양이 다릅니다. 2\~3개 챙겨가세요.

* **보조 배터리:​** 사진을 찍고 지도를 보다 보면 배터리가 빨리 닳습니다.

* **핫팩 & 얇은 겉옷 여러 겹:​** 일본의 겨울은 한국보다 덜 춥지만, 실내외 온도 차이가 큽니다. 입고 벗기 편한 옷을 챙기세요.

​

**튜터의 응원 한마디! 🌟**후쿠오카의 '다이묘 거리'에서 예쁜 카페를 가고, '캐널시티'나 '텐진'에서 쇼핑을 즐기면 동선도 짧고 완벽한 여행이 될 거예요.준비하시다가 항공권 예매 방법이나, 후쿠오카의 구체적인 카페 추천 등 더 궁금한 점이 생기면 언제든 다시 질문해 주세요! 즐거운 여행 준비 되시길 바랍니다!

# 10. 최종 선정 모델

## 선정 모델

ChatGPT (GPT-5.5)

​

## 선정 근거

1. 맞춤형 여행 추천의 정확성이 가장 우수하였다.

2. 입력된 조건을 가장 일관성 있게 반영하였다.

3. 지정한 출력 형식을 안정적으로 준수하였다.

4. 여행 준비 체크리스트가 동점인 Gemini 보다 구체적이고 실용적이었다.

# 11. 결론

본 실험에서는 동일한 프롬프트를 이용하여 세 가지 LLM의 응답 품질을 비교하였다.

평가는 정확성, 맞춤형 추천 능력, 형식 준수, 문맥 이해력, 한국어 자연스러움, 안전성, 응답 속도, 종합 만족도를 기준으로 수행하였다.

최종 선정된 모델은 이후 시스템 프롬프트 설계, Few-shot 예시 작성, 환각(Hallucination) 검증, 그리고 10턴 이상의 대화 로그 작성에 활용할 예정이다.
